# IT-портал — фронтенд (каркас)

Каркас веб-системы управления IT-инфраструктурой: HelpDesk, инвентаризация, бронирование.
UI-first, стек фронта: **React + TypeScript + Vite + Ant Design + react-i18next**.

## Запуск

```bash
npm install
npm run dev
```

Откроется на http://localhost:5173

## Что уже есть в каркасе

- **Layout + навигация** (`Sider` + `Header`), пункты меню фильтруются по роли.
- **Мок-авторизация** с переключателем роли в шапке (Сотрудник / IT-специалист / Администратор).
  Реальный вход через LDAP подключается в конце — заменит `AuthProvider`, интерфейс остаётся прежним.
- **Защита роутов по ролям** (`RoleGuard`) — разделы «Пользователи» и «Отчёты» только для админа.
- **i18n** (RU по умолчанию + EN). Переключатель языка в шапке меняет одновременно язык
  приложения и локаль Ant Design (`ConfigProvider`).
- **HelpDesk → Заявки**: таблица с фильтрами по статусу/приоритету и поиском (на мок-данных).
- Дашборд со сводными карточками; остальные разделы — заглушки.

## Структура

```
src/
  types/         типы предметной области (совпадут со схемой Prisma на бэке)
  i18n/          инициализация + локали (ru.json, en.json)
  theme/         тема Ant Design (токены, акцент)
  auth/          мок-авторизация (роль + переключение)
  components/    AppLayout, RoleGuard
  pages/         Dashboard, Tickets, заглушки остальных разделов
  App.tsx        ConfigProvider + роутинг
  main.tsx       точка входа
```
